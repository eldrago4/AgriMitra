# Required Fonts

Download and place these files in this directory before building:

1. **MaterialIcons-Regular.ttf**
   Download from: https://github.com/google/material-design-icons/tree/master/font
   Direct link: https://fonts.gstatic.com/s/materialicons/v140/flUhRq6tzZclQEJ-Vdg-IuiaDsNcIhQ8tQ.woff2
   (convert to .ttf with a font converter, or use the GitHub release)

2. **OpenSans-Regular.ttf** and **OpenSans-Semibold.ttf**
   Download from: https://fonts.google.com/specimen/Open+Sans
   Or use the .NET MAUI default fonts package.

These are referenced in MauiProgram.cs ConfigureFonts().
